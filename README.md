# 🌐 Personal Portfolio

A modern and responsive personal portfolio website built to showcase my profile, technical skills, and projects. This website serves as my online portfolio for recruiters, clients, and fellow developers.

## 🚀 Live Demo

🔗 **Website:** https://personal-portofolio-one-pi.vercel.app

## ✨ Features

* Responsive design for desktop, tablet, and mobile
* Clean and modern user interface
* About Me section
* Skills section
* Featured Projects
* Contact section
* Download CV button
* Smooth scrolling navigation

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Vercel (Deployment)

## 💻 Getting Started

1. Clone this repository.

```bash
git clone https://github.com/khusnulloh02-png/personal-portfolio.git
```

2. Open the project folder.

```bash
cd personal-portfolio
```

3. Open `index.html` in your browser or use VS Code Live Server.
